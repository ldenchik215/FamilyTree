# Руководство по исследованию родословной

Статический учебный сайт на HTML, CSS и JavaScript. Сайт подготовлен как руководство по глубокому генеалогическому исследованию: методика доказательности, источники, обработка документов, таблицы, структура цифрового архива, полезные ссылки и скачиваемый демо-проект.

## Структура

```text
project/
├─ assets/
│  └─ images/
├─ scripts/
│  ├─ data.js
│  ├─ navigation.js
│  ├─ interactions.js
│  └─ main.js
├─ downloads/
│  └─ demo_proj.zip
├─ index.html
├─ style.css
└─ README.md
```

## Запуск

Внешние библиотеки не используются. Из-за модульного JavaScript рекомендуется запускать сайт через локальный сервер:

```bash
cd project
python3 -m http.server 8000
```

Затем открыть в браузере:

```text
http://localhost:8000
```

## Что внутри демо-проекта

Файл `downloads/demo_proj.zip` содержит пример русскоязычной структуры родословного исследования:

- папки с понятными русскими названиями;
- таблицы `.xlsx` с заголовками и демо-строками;
- примеры изображений документов: ЗАГС, метрическая книга, ревизская сказка, наградной лист, семейная фотография;
- шаблоны запросов в архив и ЗАГС;
- README демо-проекта и файл контрольных сумм SHA-256.

Все имена, события и документы в демо-проекте вымышлены и нужны только для демонстрации организации архива.

## JavaScript-модули

- `scripts/data.js` — данные для карточек, источников, чек-листов и ссылок.
- `scripts/navigation.js` — мобильное оглавление и подсветка активного раздела.
- `scripts/interactions.js` — фильтры, конструктор имени файла, чек-лист, вкладки и аккордеоны.
- `scripts/main.js` — точка входа.

## Проверка

Для быстрой проверки синтаксиса JS:

```bash
node --check scripts/data.js
node --check scripts/navigation.js
node --check scripts/interactions.js
node --check scripts/main.js
```

Для визуальной проверки можно открыть сайт через локальный сервер и пройтись по разделам, кнопке скачивания и фильтрам.
